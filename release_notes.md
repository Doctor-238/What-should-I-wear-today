## 👕 오늘 뭐 입을까? (What should I wear today)

매일 아침 날씨에 맞는 완벽한 옷차림을 추천해 주고 내 옷장을 스마트하게 관리해 주는 **'오늘 뭐 입을까?'** 앱입니다! 
아래 안내에 따라 앱을 설치하고 원활한 사용을 위한 권한 및 유용한 설정을 확인해 보세요.

---

### 📥 앱 설치 방법
1. 이 릴리즈 페이지 하단의 **`오늘 뭐 입을까.apk`** 파일을 안드로이드 스마트폰에 다운로드합니다.
2. 다운로드한 APK 파일을 터치하여 실행합니다.
3. 기기 보안 설정에 따라 **'알 수 없는 앱 설치'** 경고 창이 뜬다면, 설정으로 이동하여 **'이 출처 허용'**을 활성화해 주세요.
4. 설치가 완료되면 앱을 실행합니다.

---

### 🔐 원활한 사용을 위한 권한 안내
정확한 맞춤형 추천과 편리한 앱 이용을 위해 최초 실행 시 다음 권한들의 허용이 필요합니다.

*   📍 **위치 권한**
    *   **필요한 이유:** 사용자가 현재 계신 지역의 **실시간 날씨 및 기온 정보**를 정확하게 불러오기 위해 필요합니다. 이 정보를 바탕으로 그날의 날씨에 가장 딱 맞는 옷차림을 분석하고 추천해 드립니다.
*   🔔 **알림 권한**
    *   **필요한 이유:** 매일 외출하기 전 맞춤형 옷차림 추천 알림을 보내드리거나, 갑작스러운 기상 변화 등을 놓치지 않도록 적절한 타이밍에 **푸시 알림**을 보내드리기 위해 사용됩니다.

---

### 🤖 (선택) 내 전용 API 키 설정 가이드
이 앱은 날씨 정보 제공을 위해 **OpenWeather API**를, 옷을 스캔하고 분석하기 위해 Google의 **Gemini AI**를 사용합니다. 앱 내부에는 공용으로 사용할 수 있는 '기본 키'가 내장되어 있어 **별도의 설정 없이도 바로 사용이 가능**합니다. 

단, 많은 사용자가 동시에 이용하여 **기본 키의 일일 사용량이 초과될 경우 기능이 일시적으로 제한될 수 있습니다.** 이런 경우를 대비하여 나만의 전용 키를 등록해 두시면 언제든 안정적으로 기능을 이용하실 수 있습니다.

#### 1️⃣ Gemini API 키 발급 및 등록 (AI 옷 분석)
1. **API 키 발급받기**
   - [Google AI Studio](https://aistudio.google.com/app/apikey) 웹사이트에 접속하여 구글 계정으로 로그인합니다.
   - **'Create API key'** 버튼을 눌러 새로운 프로젝트를 생성하고 키를 발급받습니다. (발급 비용은 무료입니다!)
   - 화면에 표시된 긴 문자열(API Key)을 복사합니다.
2. **앱에 등록하기**
   - '오늘 뭐 입을까?' 앱을 실행한 후 우측 하단 탭을 통해 **설정(Settings)** 화면으로 이동합니다.
   - 스크롤을 내려 **'Gemini API 키'** 항목을 찾습니다.
   - 복사해 둔 키를 입력란에 붙여넣고 **[확인]** 버튼을 누릅니다.
   - 우측 상단 상태가 **'사용자 키 사용 중'**으로 변경되었다면 등록 완료입니다!

#### 2️⃣ OpenWeather API 키 발급 및 등록 (날씨 정보)
1. **API 키 발급받기**
   - [OpenWeatherMap](https://home.openweathermap.org/users/sign_up) 웹사이트에 접속하여 계정을 생성합니다.
   - 로그인 후, 상단 메뉴의 **'My API keys'**로 이동하여 키를 발급받습니다. (무료 플랜 사용 가능)
   - 발급된 API Key 문자열을 복사합니다.
2. **앱에 등록하기**
   - 앱의 **설정(Settings)** 화면에서 **'OpenWeather API 키'** 항목을 찾습니다.
   - 복사해 둔 키를 입력란에 붙여넣고 **[확인]** 버튼을 누릅니다.
   - 우측 상단 상태가 **'사용자 키 사용 중'**으로 변경되었다면 등록 완료입니다!

*(※ 각 키는 등록 후 언제든 **[기본키로 초기화]** 버튼을 눌러 원래 상태로 되돌릴 수 있습니다.)*
